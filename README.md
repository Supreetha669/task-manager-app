# 🧩 Task Manager App

A simple full stack Python project built for the Hackathon.

## 🛠️ Tech Stack
- Backend: FastAPI
- Frontend: ReactJS
- Database: MongoDB (optional)
- Deployment: Vercel + Render
- Version Control: GitHub
